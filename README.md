# Yee — OG Generator Shortcut

Расширение для автоматического нажатия кнопки "Сгенерировать" в Персефоне горячими клавишами.

## Установка

1. `chrome://extensions/` → включить **Режим разработчика**
2. **Загрузить распакованное расширение** → выбрать папку `yee`
3. Настроить горячие клавиши: `chrome://extensions/shortcuts`
